# 🧪 QA Portfolio – OWASP Juice Shop

## 📌 About This Project

This repository contains a complete QA portfolio project based on the OWASP Juice Shop application.

The project demonstrates manual testing skills, including test case design, bug reporting, test planning, and basic security testing.

---

## 👨‍💻 About Me

I am transitioning into a Software QA role with a strong focus on manual testing.

I have experience in:
- Test case design
- Functional and negative testing
- Bug reporting
- Test planning and documentation
- Attention to detail and structured analysis

This portfolio reflects my hands-on practice and continuous learning in QA.

---

## 🧪 Project Scope

The testing process covers the following modules:

- Authentication (Login & Registration)
- Product Catalog (Main Page)
- Search Functionality
- Shopping Cart
- Checkout Process
