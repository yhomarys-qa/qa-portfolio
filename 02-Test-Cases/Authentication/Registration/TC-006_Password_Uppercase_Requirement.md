# TC-006 – Verify password uppercase letter requirement

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter a valid email address  
4. Enter a password without any uppercase letters  
5. Fill in other required fields  
6. Click on "Register"  

---

## ✅ Expected Result
- Registration is rejected  
- System displays a validation message indicating password must contain at least one uppercase letter  
- User account is not created
