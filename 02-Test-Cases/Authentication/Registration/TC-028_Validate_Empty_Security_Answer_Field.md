# TC-028 - Validate Empty Security Answer Field

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Leave the "Security Answer" field empty
8. Click on the "Register" button.

---

## ✅ Expected Result
- A validation message should be displayed indicating that the security answer field is required
- Registration should not be allowed
