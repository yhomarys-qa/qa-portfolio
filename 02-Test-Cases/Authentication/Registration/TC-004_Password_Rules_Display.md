# TC-004 – Verify password rules display during registration

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Focus on the password field or enable password hint/help option (if available)  
4. Observe the password requirements displayed on the screen  

---

## ✅ Expected Result
- System displays all password requirements (e.g. minimum length, complexity rules)  
- User is informed about valid password criteria before or during input
