# TC-002 – Verify registration with existing email

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter an email address that already exists in the system  
4. Fill in the remaining required fields  
5. Click on "Register"  

---

## ✅ Expected Result
- Registration is rejected  
- Error message is displayed indicating the email already exists  
- User account is not created
