# TC-010 – Verify password confirmation match

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter a valid email address  
4. Enter a valid password in the "Password" field  
5. Enter a different password in the "Repeat Password" field  
6. Fill in other required fields  
7. Click on "Register"  

---

## ✅ Expected Result
- Registration is rejected  
- System displays a validation message indicating passwords do not match  
- User account is not created
