# TC-014 - Validate Password Maximum Length (40 Characters)

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a password with exactly 40 characters
5. Fill in all required fields
6. Click on the "Register" button

---

## ✅ Expected Result

The system should accept the password and allow the registration process to continue
