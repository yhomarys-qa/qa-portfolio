# TC-034 - Validate Security Answer With Uppercase And Lowercase Characters

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Enter uppercase and lowercase characters in the "Security Answer" field
8. Click on the "Register" button

---

## ✅ Expected Result

The system should accept uppercase and lowercase characters in the "Security Answer" field and allow the registration process to continue
