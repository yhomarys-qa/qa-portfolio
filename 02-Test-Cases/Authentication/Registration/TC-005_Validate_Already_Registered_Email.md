# TC-005 – Validate Already Registered Email

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter an email that is already registered in the system
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Enter a valid security answer
8. Click on the "Register" button

---

## ✅ Expected Result

The system should prevent the registration process and display a message indicating that the email is already in use
