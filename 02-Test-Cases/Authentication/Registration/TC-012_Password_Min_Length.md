# TC-012 – Verify password minimum length requirement

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter a valid email address  
4. Enter a password with less than 8 characters  
5. Fill in other required fields  
6. Click on "Register"  

---

## ✅ Expected Result

System displays a validation message indicating password must meet minimum length requirement
