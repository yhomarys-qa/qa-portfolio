# TC-024 – Verify security question selection during registration

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter a valid email address  
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field  
6. Select a security question from the available options  
7. Fill in other required fields  
8. Click on "Register"  

---

## ✅ Expected Result
- Security question is saved in the account profile  
- Security question is available for account recovery process
