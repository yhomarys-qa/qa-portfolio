# TC-023 - Validate Empty Security Question Field

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Leave the "Security Question" field blank
7. Fill in other required fields 
8. Click on the "Register" button

---

## ✅ Expected Result

A validation message should be displayed indicating that the security question field is required
