# TC-017 - Validate Empty Repeat Password Field

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password in the "Password" field
5. Leave the "Repeat Password" field empty
6. Fill in other required fields  
7. Click on the "Register" button

---

## ✅ Expected Result
- A validation message should be displayed indicating that the "Repeat Password" field is required
- Registration should not be allowed
