# TC-035 - Validate Security Answer Maximum Length (255 Characters)

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Enter exactly 255 characters in the "Security Answer" field
8. Click on the "Register" button

---

## ✅ Expected Result

The system should accept a security answer containing exactly 255 characters and allow the registration process to continue
