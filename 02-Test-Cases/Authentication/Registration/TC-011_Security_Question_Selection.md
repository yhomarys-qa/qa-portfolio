# TC-011 – Verify security question selection during registration

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter a valid email address  
4. Enter a valid password and confirm it  
5. Select a security question from the available options  
6. Provide an answer to the selected security question  
7. Fill in other required fields  
8. Click on "Register"  

---

## ✅ Expected Result
- User is successfully registered  
- Security question is saved in the account profile  
- Security question is available for account recovery process
