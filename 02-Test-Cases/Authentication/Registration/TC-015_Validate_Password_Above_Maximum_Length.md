# TC-015 - Validate Password Above Maximum Length (40 Characters)

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a password with more than 40 characters
5. Fill in all required fields
6. Click on the "Register" button

---

## ✅ Expected Result
- A validation message should be displayed indicating that the password exceeds the maximum allowed length of 40 characters
- Registration should not be allowed
