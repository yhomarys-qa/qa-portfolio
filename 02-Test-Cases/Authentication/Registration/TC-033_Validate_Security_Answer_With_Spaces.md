# TC-033 - Validate Security Answer With Spaces

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Enter a value containing spaces in the "Security Answer" field (e.g., "Maria Clara Perez")
8. Click on the "Register" button

---

## ✅ Expected Result

The system should accept spaces in the "Security Answer" field and allow the registration process to continue
