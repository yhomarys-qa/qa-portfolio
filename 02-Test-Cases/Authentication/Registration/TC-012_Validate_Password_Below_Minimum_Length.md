# TC-012 – Validate Password Below Minimum Length (8 Characters)

## 🧪 Test Steps
1. Open the application.
2. Access the registration page.
3. Enter a valid email address.
4. Enter a password with less than 8 characters (e.g., "Pass12").
5. Confirm the password in the "Repeat Password" field.
6. Select a security question.
7. Enter a valid security answer.
8. Click on the "Register" button.

## ✅ Expected Result

A validation message should be displayed indicating that the password must contain at least 8 characters. Registration should not be allowed
