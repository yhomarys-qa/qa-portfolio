# TC-010 – Validate Password Accepts Special Characters Only

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a password using only special characters (e.g., "@@@###$$$")
5. Fill in all required fields
6. Click on the "Register" button

---

## ✅ Expected Result

The system should validate whether special-character-only passwords are allowed according to password policy rules
