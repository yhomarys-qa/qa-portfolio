# TC-019 - Validate The Same Password In The "Repeat Password" Field

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password in the "Password" field
5. Enter the same password in the "Repeat Password" field
6. Fill in other required fields 
7. Click on the "Register" button

---

## ✅ Expected Result

The system should accept the matching passwords and allow the registration process to continue
