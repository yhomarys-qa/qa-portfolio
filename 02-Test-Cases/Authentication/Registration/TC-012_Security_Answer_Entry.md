# TC-012 – Verify security answer entry during registration

## 🧪 Test Steps
1. Open the application  
2. Navigate to the Registration page  
3. Enter a valid email address  
4. Enter a valid password in the "Password" field
5. Enter the same password in the "Repeat Password" field 
6. Select a security question  
7. Enter a valid answer in the security answer field    
8. Click on "Register"  

---

## ✅ Expected Result
- User is successfully registered  
- Security answer is saved in the account profile  
- Security answer is available for account recovery process
