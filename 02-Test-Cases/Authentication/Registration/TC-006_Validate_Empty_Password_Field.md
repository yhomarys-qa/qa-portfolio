# TC-006 – Validate Empty Password Field

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Leave the "Password" field empty
5. Enter a value in the "Repeat Password" field
6. Select a security question
7. Enter a valid security answer
8. Click on the "Register" button

---

## ✅ Expected Result

The system should prevent the registration process and display a validation message indicating that the password field is required
