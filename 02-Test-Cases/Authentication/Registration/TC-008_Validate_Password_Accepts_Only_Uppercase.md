# TC-008 – Validate Password Accepts Only Uppercase Characters

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a password using only uppercase letters (e.g., "PASSWORD")
5. Confirm the password in the "Repeat Password" field.
6. Select a security question.
7. Enter a valid security answer.
8. Click on the "Register" button.

---

## ✅ Expected Result

The system should validate whether uppercase-only passwords are allowed according to password policy rules
