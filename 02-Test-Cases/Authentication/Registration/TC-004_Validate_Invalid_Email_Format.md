# TC-004 – Validate Invalid Email Format

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter an invalid email format (e.g., "useremail.com" or "user@com")
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Enter a valid security answer
8. Click on the "Register" button

## ✅ Expected Result

The system should prevent the registration process and display a validation message indicating that the email format is invalid
