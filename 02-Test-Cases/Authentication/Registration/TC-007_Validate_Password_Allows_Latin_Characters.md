# TC-007 – Validate Password Allows Latin Characters

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a password containing Latin characters (e.g., "PasswordTest")
5. Confirm the password in the "Repeat Password" field.
6. Select a security question.
7. Enter a valid security answer.
8. Click on the "Register" button.

---

## ✅ Expected Result

The system should accept passwords containing Latin characters and proceed according to validation rules
