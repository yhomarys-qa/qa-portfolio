# TC-034 - Validate Security Answer Minimum Length

## 🧪 Test Steps
1. Open the application
2. Access the registration page
3. Enter a valid email address
4. Enter a valid password
5. Enter the same password in the "Repeat Password" field
6. Select a security question
7. Enter a single character in the "Security Answer" field
8. Click on the "Register" button

---

## ✅ Expected Result

The system should accept a minimum length of one character in the "Security Answer" field and allow the registration process to continue
