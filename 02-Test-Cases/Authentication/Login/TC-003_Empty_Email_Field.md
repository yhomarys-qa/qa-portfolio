# TC-003 – Verify that the system rejects login attempts with an empty Email field

## 🧪 Test Steps
1. Open the application  
2. Navigate to the login page  
3. Leave the Email field empty  
4. Enter an valid password in the Password field  
5. Click on "Log in"  

---

## ✅ Expected Result

A validation message should be displayed indicating that the email field is required
