# TC-003 – Verify that login fails when the Email field is left empty

## 🧪 Test Steps
1. Open the application  
2. Navigate to the login page  
3. Leave the Email field empty  
4. Enter an invalid password in the Password field  
5. Click on "Log in"  

---

## ✅ Expected Result

A validation message indicating that the Email field is required should be displayed
