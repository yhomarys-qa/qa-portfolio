# TC-019 - Verify that the tooltip/help text is displayed for the Security Question field

## Test Steps
1. Navigate to the "Forgot Password" form
2. Hover the mouse cursor over the question mark icon located at the end of the Security Question field
3. Observe the tooltip/help text displayed

---

## ✅ Expected Result
- The tooltip/help text should be displayed successfully
- The message "Please answer your security question." should be visible to the user
