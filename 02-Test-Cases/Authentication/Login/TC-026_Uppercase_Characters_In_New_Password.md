# TC-026 - Verify that the New Password field accepts uppercase letters

## Test Steps
1. Navigate to the "Forgot Password" form
2. Enter a valid email address
3. Enter a valid answer in the "Security Question" field
4. Enter a password containing uppercase letters in the New Password field (e.g., TesTing) 
5. Observe the validation behavior

---

## Expected Result
- The password should accept uppercase letters successfully
- No validation error related to uppercase letters should be displayed
