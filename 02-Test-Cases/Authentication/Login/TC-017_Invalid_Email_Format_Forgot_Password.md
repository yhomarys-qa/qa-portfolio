# TC-017 - Verify system behavior when an invalid email address is entered in the Forgot Password form

## Test Steps
1. Navigate to the Forgot Password form.
2. Enter an invalid email address in the Email field.
3. Observe the remaining form fields and validation behavior.

---

## Expected Result
- The remaining form fields should remain blocked or disabled.
- The system should display an appropriate validation message indicating that the email format is invalid.
- The user should not be allowed to continue the password recovery process.
