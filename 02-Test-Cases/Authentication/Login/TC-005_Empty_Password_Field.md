# TC-005 – Check if the system rejects the login when the Password field is empty

## 🧪 Test Steps
1. Open the application  
2. Navigate to the login page  
3. Enter an valid email in the email field
4. Leave the Password field empty
5. Click on "Log in"  

---

## ✅ Expected Result

A validation message indicating that the Password field is required should be displayed
