# TC-012 – Verify "Remember Me" functionality

## 🧪 Test Steps
1. Open the application  
2. Navigate to the login page  
3. Check the "Remember me" option  
4. Enter valid email and password  
5. Click on "Log in"  
6. Close the browser  
7. Reopen the application  

---

## ✅ Expected Result
- User session remains active after reopening the browser  
- User does not need to log in again
