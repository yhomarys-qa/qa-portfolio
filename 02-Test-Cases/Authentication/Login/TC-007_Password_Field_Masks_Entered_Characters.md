# TC-007 – Verify that characters entered in the Password field are masked

## 🧪 Test Steps
1. Open the application  
2. Navigate to the login page  
3. Enter any password in the Password field
4. Observe the characters displayed in the Password field  

---

## ✅ Expected Result
- All characters entered in the Password field should be masked
- The password should not be visible in plain text
