# TC-006 – Verify that the system rejects login attempts with an invalid password

## 🧪 Test Steps
1. Open the application  
2. Navigate to the login page  
3. Enter an valid email in the email field   
4. Enter an invalid password in the Password field  
5. Click on "Log in"  

---

## ✅ Expected Result

Validation message "Invalid email or password" is displayed
