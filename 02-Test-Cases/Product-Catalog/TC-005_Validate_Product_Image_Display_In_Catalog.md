# TC-005 – Validate Product Image Display In Catalog

## 🧪 Test Steps
1. Open the application
2. Log in with valid user credentials
3. Access the product catalog page
4. Observe the product cards displayed in the catalog
5. Verify that each product card displays a product image

---

## ✅ Expected Result

Each product card in the catalog should display the corresponding product image correctly and visibly
