# TC-010 – Validate Product Comments Display Without Login

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Observe the comments section displayed for the selected product

---

## ✅ Expected Result

The system should display product comments correctly, including usernames and comment content, without requiring user authentication
