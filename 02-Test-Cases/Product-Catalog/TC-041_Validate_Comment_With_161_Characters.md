# TC-041 - Validate Comment With 161 Characters

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Navigate to any product
4. Enter a comment with 161 characters
5. Attempt to submit the comment

---

## ✅ Expected Result

The system should prevent the comment from being submitted when it exceeds 160 characters, and the submission action is blocked
