# TC-007 – Validate Product Like And Comment Functionality Without Login

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product
5. Attempt to interact with the like functionality
6. Attempt to add or interact with the comments section
7. Observe the system behavior

---

## ✅ Expected Result

The system should handle like and comment interactions correctly for non-authenticated users according to the application rules
