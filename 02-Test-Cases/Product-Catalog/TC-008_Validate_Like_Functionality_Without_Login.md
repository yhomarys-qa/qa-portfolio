# TC-008 – Validate Like Functionality Without Login

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Click on the like icon
6. Observe the system behavior

---

## ✅ Expected Result

The system should not allow non-authenticated users to interact with the like functionality
