# TC-006 – Validate Product Reviews Display Without Login

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Observe the reviews section displayed for the selected product  

---

## ✅ Expected Result

The system should display the product reviews correctly, including review information and the number of reviews, without requiring user authentication
