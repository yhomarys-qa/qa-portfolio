# TC-007 – Validate Product Search Functionality

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Enter a valid product name in the search field
5. Execute the search action
6. Observe the displayed search results

---

## ✅ Expected Result

The system should display products related to the searched product name correctly
