# TC-035 - Validate Comment Send Button State Based On Input

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Navigate to any product
4. Observe the [Enviar] button without typing a comment
5. Enter valid comment text (within limits)
6. Observe the button state changes

---

## ✅ Expected Result

The system should keep the [Enviar] button disabled when no valid input is provided and enable it when a valid comment is entered
