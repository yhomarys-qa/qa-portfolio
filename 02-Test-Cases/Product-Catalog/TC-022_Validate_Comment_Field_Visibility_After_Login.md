# TC-022 - Validate Comment Field Visibility After Login

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Locate the comment section
6. Observe the comment input field

---

## ✅ Expected Result

The system should display the comment input field correctly and visibly for authenticated users
