# TC-038 - Validate Comment With Special Characters

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Enter a comment containing special characters (e.g., "Excellent product! :-)")
6. Submit the comment

---

## ✅ Expected Result

The system should handle comments containing special characters in accordance with the application's rules
