# TC-013 - Validate [Close] Button Functionality For Unauthenticated User

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Observe the product details screen
6. Click on the [Fechar] button

---

## ✅ Expected Result

The system should close the product details screen successfully when the [Fechar] button is clicked for unauthenticated users
