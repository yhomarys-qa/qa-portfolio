# TC-012 - Validate [Close] Button Visibility For Unauthenticated User

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Observe the product details screen
6. Locate the [Fechar] button

---

## ✅ Expected Result

The system should display the [Fechar] button correctly for unauthenticated users and allow them to close the product details screen
