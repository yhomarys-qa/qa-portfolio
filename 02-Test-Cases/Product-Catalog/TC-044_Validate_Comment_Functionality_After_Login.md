# TC-044 - Validate Comment Functionality After Login

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Enter a valid comment in the comment field (e.g., "This Product Is Amazing and very useful, I bought 3 items and received them in 2 days :-)")
6. Submit the comment

---

## ✅ Expected Result

The system should allow authenticated users to submit comments successfully
