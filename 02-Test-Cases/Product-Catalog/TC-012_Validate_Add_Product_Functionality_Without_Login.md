# TC-012 - Validate Add Product Functionality Without Login

## 🧪 Test Steps
1. Open the application
2. Do not log in
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Attempt to add the selected product to the basket

---

## ✅ Expected Result

The system should not provide the option to add products to the basket for non-authenticated users
