# TC-012 - Validate Comment With Latin Characters

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Enter a comment using Latin characters in the comment field
6. Submit the comment
7. Observe the system behavior

---

## ✅ Expected Result

The system should accept comments containing Latin characters correctly.
