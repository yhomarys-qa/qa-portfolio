# TC-005 – Validate Product Card Elements Display

## 🧪 Test Steps
1. Open the application
2. Access the product catalog page
3. Observe the product cards displayed in the catalog
4. Verify the elements displayed in each product card

---

## ✅ Expected Result
Each product card should display the following elements correctly:
- Product image
- Product name
- Product price
- Product rating or likes (if applicable)
