# TC-010 – Verify price currency format based on locale

## 🧪 Test Steps
1. Open the application  
2. Change the application language or locale (if available)  
3. Navigate to the Product Catalog page  
4. Observe the price format of products  

---

## ✅ Expected Result
- Prices are displayed using the correct currency symbol for the selected locale  
- Number formatting matches the locale (e.g., decimal and thousand separators)  
- Currency format is consistent across all products
