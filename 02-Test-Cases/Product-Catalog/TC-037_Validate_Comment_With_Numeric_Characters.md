# TC-037 - Validate Comment With Numeric Characters

## 🧪 Test Steps
1. Open the application
2. Log in with valid credentials
3. Click on the application icon in the top navigation bar
4. Click on any product displayed in the catalog
5. Enter a comment containing numeric characters (e.g., "I bought 2 items and received them in 3 days")
6. Submit the comment

---

## ✅ Expected Result

The system should accept comments containing numeric characters successfully
