# TC-005 – Validate Product Quantity Controls Are Visible

## 🧪 Test Steps
1. Open the application
2. Log in with valid user credentials
3. Add a product to the shopping cart
4. Navigate to the shopping cart page
5. Verify the visibility of the "+" and "-" quantity control icons

---

## ✅ Expected Result

The system should display the "+" and "-" quantity control icons along with the current product quantity
