# TC-019 - Validate Bonus Points Message For Eligible Order

## 🧪 Test Steps
1. Open the application
2. Log in with valid user credentials
3. Add five or more products to the shopping cart
4. Navigate to the shopping cart page
5. Verify the bonus points message displayed below the "Checkout" button

---

## ✅ Expected Result

The system should display the message indicating that the user will gain 500 bonus points from the order
