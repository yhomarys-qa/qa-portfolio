# TC-007 – Validate Product Quantity Update

## 🧪 Test Steps
1. Open the application
2. Log in with valid user credentials
3. Add a product to the shopping cart
4. Navigate to the shopping cart page
5. Change the quantity of the added product
6. Save or update the cart if applicable
---

## ✅ Expected Result

The system should successfully update the product quantity and reflect the updated value in the shopping cart
