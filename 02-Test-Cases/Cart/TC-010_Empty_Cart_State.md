# TC-010 – Verify empty cart state

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Navigate to the cart page without adding any products  
4. Add a product to the cart  
5. Remove the product from the cart  
6. Observe the cart page  

---

## ✅ Expected Result

Cart displays an empty state message (e.g., "Your basket is empty")
