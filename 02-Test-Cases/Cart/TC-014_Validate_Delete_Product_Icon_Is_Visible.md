# TC-014 - Validate Delete Product Icon Is Visible

## 🧪 Test Steps
1. Open the application
2. Log in with valid user credentials
3. Add a product to the shopping cart
4. Navigate to the shopping cart page
5. Verify the visibility of the delete/trash icon for the added product

---

## ✅ Expected Result

The delete/trash icon should be visible and accessible for the added product in the shopping cart
