# TC-018 - Validate Bonus Points Message For Empty Cart

## 🧪 Test Steps
1. Open the application
2. Log in with valid user credentials
3. Navigate to the shopping cart page without adding any products
4. Verify the bonus points message displayed below the "Checkout" button

---

## ✅ Expected Result

The system should display the message indicating that the user will gain 0 bonus points from the order
