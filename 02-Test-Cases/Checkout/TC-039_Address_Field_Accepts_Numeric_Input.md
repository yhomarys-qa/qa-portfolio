# TC-039 – Verify Address field accepts numeric input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter a valid mobile phone number
5. Enter a valid ZIP code
6. EEnter only the numeric value in the address field (e.g., 123456)       
7. Fill all other required fields  
8. Click on "Submit"

---

## ✅ Expected Result

- Numeric input is accepted in the Address field  
- No validation error is displayed
