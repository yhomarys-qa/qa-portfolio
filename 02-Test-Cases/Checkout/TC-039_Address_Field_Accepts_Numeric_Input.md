# TC-039 – Verify Address field accepts numeric input

## 🧪 Test Steps

1. Open Add New Address form  
2. Enter valid data in all required fields except Address  
3. Locate the "Address" field  
4. Enter numeric input only (e.g., 123456)  
5. Click on "Submit"  

---

## ✅ Expected Result

- Numeric input is accepted in the Address field  
- No validation error is displayed  
- Form submission proceeds successfully
