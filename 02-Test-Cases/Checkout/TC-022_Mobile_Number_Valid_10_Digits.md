# TC-022 – Verify Mobile Number accepts 10-digit value

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name 
3. Enter a valid name
4. Enter a 10-digit number in the "Mobile number" field (e.g., 1234567890)
5. Fill all other required fields
6. Click on "Submit"

---

## ✅ Expected Result
- Input is accepted  
- Field remains valid
