# TC-016 – Verify Name field rejects numeric input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name (e.g., "Brazil")
3. Enter numeric value in Name field (e.g., "12345")
4. Fill all other fields correctly  
5. Click on "Submit"

---

## ✅ Expected Result
- Numeric input is rejected  
- Validation message is displayed  
- Field is marked as invalid
