# TC-015 – Verify Name field accepts full names with spaces

## 🧪 Test Steps

2. Enter a valid country name 
3. Enter the full name, including spaces, in the name field (e.g., "Maria Silva")
4. Fill all other required fields  
5. Click on "Submit".
