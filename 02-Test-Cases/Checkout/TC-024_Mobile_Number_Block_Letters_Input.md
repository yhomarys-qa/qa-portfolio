# TC-024 – Verify Mobile Number blocks alphabetic input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Introduce alphabetic characters into the Mobile Number field (e.g., abcdef)  
5. Fill all other required fields  
6. Click on "Submit"  

---

## ❌ Expected Result
- Alphabetic characters are not accepted  
- No characters appear in the input field  
- System prevents invalid keystrokes
