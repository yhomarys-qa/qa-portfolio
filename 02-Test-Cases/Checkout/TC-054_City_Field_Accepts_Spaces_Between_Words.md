# TC-054 – Verify City field accepts spaces between words

## 🧪 Test Steps

1. Open Add New Address form  
2. Fill all required fields with valid data except City  
3. Locate the "City" field  
4. Enter a city name containing spaces (e.g., New York)  
5. Click on "Submit"  

---

## ✅ Expected Result

- City names containing spaces are accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
