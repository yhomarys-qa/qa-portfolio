# TC-064 – Verify State field accepts single-character input

## 🧪 Test Steps

1. Open Add New Address form  
2. Fill all required fields with valid data except State  
3. Locate the "State" field  
4. Enter a single-character State value (e.g., A)  
5. Click on "Submit"  

---

## ✅ Expected Result

- Single-character input is accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
