# TC-044 – Verify Address field accepts maximum boundary value (160 characters)

## 🧪 Test Steps

1. Open Add New Address form  
2. Enter valid data in all required fields except Address  
3. Locate the "Address" field  
4. Enter an Address value with exactly 160 characters  
5. Click on "Submit"

## ✅ Expected Result

- Address input is accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
