# TC-044 – Verify Address field accepts maximum boundary value (160 characters)

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter a valid mobile phone number
5. Enter a valid ZIP code
6. Enter an Address value with exactly 160 characters        
7. Fill all other required fields  
8. Click on "Submit"

---

## ✅ Expected Result
- Address input is accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
