# TC-025 – Verify Mobile Number allows '+' and '-' characters

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter special characters in the Mobile Number field (e.g., +, -) using the value +55-11-98765-4321
5. Fill all other required fields  
6. Click on "Submit"

---

## ✅ Expected Result
- '+' and '-' characters are accepted  
- Input is not blocked  
- System applies validation rules after input
