# TC-038 – Verify Address field accepts alphabetic input

## 🧪 Test Steps

1. Open Add New Address form  
2. Enter a valid Country  
3. Enter a valid Name  
4. Enter a valid Mobile Number  
5. Enter a valid ZIP Code  
6. Locate the "Address" field  
7. Enter alphabetic input only (e.g., Rua das Flores)  
8. Fill all remaining required fields with valid data  
9. Click on "Submit"  

---

## ✅ Expected Result

- Alphabetic input is accepted in the Address field  
- No validation error message is displayed  
- Form submission proceeds successfully
