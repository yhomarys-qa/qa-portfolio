# TC-009 – Verify Country field validation when left empty

## 🧪 Test Steps
1. Open the application  
2. Navigate to Add New Address form  
3. Leave "Country" field empty  
4. Fill all other required fields  
5. Click on "Submit"  

---

## ✅ Expected Result
- Country field is highlighted in red  
- Error message "Please provide a country" is displayed  
- Form is not submitted
