# TC-035 – Verify ZIP Code maximum length validation

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter a valid mobile phone number
5. Enter a valid ZIP Code with exactly 8 characters (e.g., 12304678)     
6. Fill all other required fields  
7. Click on "Submit"

## ✅ Expected Result

- No validation error is displayed
