# TC-020 – Verify Mobile Number field accepts valid numeric input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name 
4. Enter a valid mobile number (e.g., "11987654321")
5. Fill all other required fields
6. Click on "Submit"

---

## ✅ Expected Result
- Mobile number is accepted  
- No validation errors are shown  
- Field remains valid
