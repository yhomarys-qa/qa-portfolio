# TC-008 - Validate Country Field With Alphanumeric Characters

## 🧪 Test Steps
1. Open Add New Address form  
2. Enter alphanumeric characters in the "Country" field (e.g., "Br4z1l123") 
3. Fill all other required fields  
4. Click on "Submit"  

---

## ✅ Expected Result

The system should display a validation message indicating that the "Country" field does not accept alphanumeric characters and only valid country names are allowed.
