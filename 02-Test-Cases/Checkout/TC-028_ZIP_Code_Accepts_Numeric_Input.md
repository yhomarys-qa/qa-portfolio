# TC-028 – Verify ZIP Code accepts numeric input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter a valid mobile phone number
5. Enter a valid numeric value in the ZIP code field. (e.g., 10001) 
6. Fill all other required fields  
7. Click on "Submit"

---

## ✅ Expected Result

ZIP Code accepts numeric input
