# TC-028 – Verify ZIP Code accepts numeric input

## 🧪 Test Steps
1. Open Add New Address form  
2. Locate the "ZIP Code" field  
3. Enter a valid numeric value (e.g., 12345678)  
4. Fill all other required fields with valid data  
5. Click on "Submit"  

---

## ✅ Expected Result
- ZIP Code accepts numeric input  
- No validation error is displayed  
- Field value is preserved correctly  
- Form proceeds to next step (if all fields are valid)
