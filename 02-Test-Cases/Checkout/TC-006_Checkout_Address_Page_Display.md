# TC-006 – Verify user can view and interact with address selection

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Add a product to the cart  
4. Navigate to the cart page  
5. Click on "Checkout"  
6. Attempt to select an address (if available)  
7. Attempt to click "Add Address"  

---

## ✅ Expected Result
- Address selection page is displayed  
- User can interact with available addresses  
- "Add Address" button is clickable  
- No UI or interaction issues occur
