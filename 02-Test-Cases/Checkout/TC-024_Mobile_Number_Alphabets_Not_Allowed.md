# TC-024 – Verify Mobile Number rejects alphabetic input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter letters (e.g., abcdef)   
5. Fill all other required fields  
6. Click on "Submit"  

---

## ❌ Expected Result
- Input is rejected  
- Validation error is shown
