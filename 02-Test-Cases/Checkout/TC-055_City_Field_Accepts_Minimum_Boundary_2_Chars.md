# TC-055 – Verify City field accepts minimum boundary value (2 characters)

## 🧪 Test Steps

1. Open Add New Address form  
2. Fill all required fields with valid data except City  
3. Locate the "City" field  
4. Enter a City value with exactly 2 characters (e.g., NY)  
5. Click on "Submit"  

---

## ✅ Expected Result

- Two-character City input is accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
