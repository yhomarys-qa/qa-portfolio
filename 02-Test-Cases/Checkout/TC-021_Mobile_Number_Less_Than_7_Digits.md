# TC-021 – Verify Mobile Number rejects less than 7 digits

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name 
3. Enter a valid name
4. Enter 4-digit number (e.g., 1234)
5. Fill all other required fields
6. Click on "Submit"

---

## ✅ Expected Result
- Error message is displayed  
- Message: "O número de celular deve corresponder ao formato 1000000-9999999999"
- Field is highlighted in red
