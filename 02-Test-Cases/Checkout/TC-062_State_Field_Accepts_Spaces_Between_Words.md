# TC-062 – Verify State field accepts spaces between words

## 🧪 Test Steps

1. Open Add New Address form  
2. Fill all required fields with valid data except State  
3. Locate the "State" field  
4. Enter a State value containing spaces  
   (e.g., Rio Grande do Sul)  
5. Click on "Submit"  

---

## ✅ Expected Result

- State names containing spaces are accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
