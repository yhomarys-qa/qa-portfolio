# TC-018 – Verify Mobile Number field validation when left empty

## 🧪 Test Steps
1. Open Add New Address form  
2. Leave "Mobile Number" field empty  
3. Fill all other required fields correctly  
4. Click on "Submit"  

---

## ✅ Expected Result
- Mobile Number field is highlighted in red  
- Validation message is displayed (e.g., "Please provide a mobile number")  
- Form is not submitted
