# TC-005 – Verify checkout process with products

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Add one or more products to the cart  
4. Navigate to the cart page  
5. Click on "Checkout"  

---

## ✅ Expected Result
- User is redirected to the "Select an address" page  
- Existing addresses (if any) are displayed  
- Option to add a new address is available  
- "Add Address" button is visible and clickable  
- No errors occur during navigation
