# TC-041 – Verify Address field accepts valid special characters

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter a valid mobile phone number
5. Enter a valid ZIP code
6. Enter valid special characters in the Address field (e.g., Street A-12, Block #5)       
7. Fill all other required fields  
8. Click on "Submit"

---

## ✅ Expected Result
- Valid special characters are accepted  
- No validation error message is displayed
