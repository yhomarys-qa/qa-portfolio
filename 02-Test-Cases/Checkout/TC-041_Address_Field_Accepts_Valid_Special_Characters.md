# TC-041 – Verify Address field accepts valid special characters

## 🧪 Test Steps

1. Open Add New Address form  
2. Enter valid data in all required fields except Address  
3. Locate the "Address" field  
4. Enter valid special characters in the Address field (e.g., Street A-12, Block #5)  
5. Click on "Submit"  

---

## ✅ Expected Result

- Valid special characters are accepted  
- No validation error message is displayed  
- Form submission proceeds successfully
