# TC-010 - Validate Country Field With Uppercase Characters

## 🧪 Test Steps
1. Open Add New Address form 
2. Enter a valid country name using only uppercase characters (e.g., "BRAZIL")
3. Fill other fields with valid data
4. Click on "Submit"

---

## ✅ Expected Result

The system should accept uppercase characters in the "Country" field and correctly process or normalize the value according to system rules
