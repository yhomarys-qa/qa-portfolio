# TC-026 – Verify Mobile Number rejects special characters

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter special characters in the Mobile Number field (e.g., @@@###)  
5. Fill all other required fields  
6. Click on "Submit"
  
---

## ❌ Expected Result
- Special characters are not accepted  
- No characters appear in the input field  
- System prevents invalid keystrokes
