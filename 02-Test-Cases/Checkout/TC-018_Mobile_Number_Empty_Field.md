# TC-018 – Verify Mobile Number field cannot be empty

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Leave "Mobile Number" field empty  
5. Fill all other required fields  
6. Click on "Submit"  

---

## ✅ Expected Result
- Mobile Number field is highlighted in red  
- Validation message is displayed (e.g., "Please provide a mobile number")
