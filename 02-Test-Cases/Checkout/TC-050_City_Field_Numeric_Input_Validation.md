# TC-050 – Verify City field behavior with numeric input

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter a valid mobile phone number
5. Enter a valid ZIP code
6. Enter a valid address
7. Enter numeric input only (e.g., 123456)  
8. Fill all other required fields  
9. Click on "Submit"

---

## ✅ Expected Result
- System behavior follows validation rules:
  - numeric input is rejected, or
  - validation error message is displayed
