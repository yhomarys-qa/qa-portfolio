# TC-027 – Verify ZIP Code cannot be empty

## 🧪 Test Steps
1. Open Add New Address form
2. Enter a valid country name
3. Enter a valid name
4. Enter valid special characters
5. Leave ZIP Code field empty 
6. Fill all other required fields  
7. Click on "Submit"

---

## ❌ Expected Result
- Error message displayed: "Por favor informe um código postal"  
- Field highlighted in red
