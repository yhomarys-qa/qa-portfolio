# TC-034 – Verify ZIP Code accepts maximum boundary value (8 characters)

## 🧪 Test Steps

1. Open Add New Address form  
2. Locate the "ZIP Code" field  
3. Enter a valid ZIP Code with exactly 8 characters (e.g., 12345678)  
4. Fill all other required fields with valid data  
5. Click on "Submit"  

---

## ✅ Expected Result

- ZIP Code is accepted  
- No validation error is displayed  
- Form submission proceeds successfully
