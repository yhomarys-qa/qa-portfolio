# TC-004 – Verify search with uppercase input

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a product name in uppercase (e.g., "APPLE")  
5. Press "Enter" or click on the search icon  

---

## ✅ Expected Result
- Relevant products matching the search term are displayed
- Results correspond to the entered product name
