# TC-003 – Verify search with partial product name

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a partial product name (e.g., "app" for "Apple")  
5. Press "Enter" or click on the search icon  

---

## ✅ Expected Result
- Products containing the partial search term are displayed  
- Results include all relevant matches  
- No unrelated products are shown
