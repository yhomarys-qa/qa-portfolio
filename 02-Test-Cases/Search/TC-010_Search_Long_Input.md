# TC-010 – Verify search with very long input

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a very long string (e.g., 20+ characters)  
5. Press "Enter"  

---

## ✅ Expected Result
- No matching results are displayed  
- A "No results found" message is shown
