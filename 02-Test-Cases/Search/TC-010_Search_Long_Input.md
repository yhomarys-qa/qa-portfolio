# TC-010 – Verify search with very long input

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a very long string (e.g., 200+ characters)  
5. Press "Enter"  

---

## ✅ Expected Result
- System handles input without crashing  
- Search executes or safely returns no results  
- No performance issues or UI breakage occur
