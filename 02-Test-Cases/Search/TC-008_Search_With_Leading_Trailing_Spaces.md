# TC-008 – Verify search with leading and trailing spaces

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a valid product name with spaces (e.g., "   apple   ")  
5. Press "Enter"  

---

## ✅ Expected Result
- Leading and trailing spaces are ignored  
- Search input is trimmed automatically  
- Relevant products are displayed correctly
- Results are identical to searching without extra spaces (e.g., "apple")  
- No errors or unexpected behavior occur
