# TC-004 – Verify case insensitive search

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a product name in uppercase (e.g., "APPLE")  
5. Press "Enter" or click on the search icon  
6. Repeat the search using lowercase (e.g., "apple")  

---

## ✅ Expected Result
- Search results are the same regardless of text case  
- Relevant products are displayed for both inputs  
- No errors occur during the search
