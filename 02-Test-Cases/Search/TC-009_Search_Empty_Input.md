# TC-009 – Verify search with empty input

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Leave the search field empty  
5. Press "Enter"  

---

## ✅ Expected Result
- System does not perform invalid search  
- Either full product list is displayed or no action occurs  
- No error message or crash occurs
