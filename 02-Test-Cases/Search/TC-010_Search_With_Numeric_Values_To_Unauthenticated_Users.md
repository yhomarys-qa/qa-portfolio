# TC-00510 - Search with Numeric Values to Unauthenticated Users

## 🧪 Test Steps
1. Open the search field
2. Enter numeric values in the search field (example: "123" or "89.99")
3. Click the search button

---

## ✅ Expected Result

The system must execute the search and display a "No results found" message when no products are associated with numeric values
