# TC-004 - Items Per Page Filter Displayed in Search Results to Unauthenticated Users

## 🧪 Test Steps
1. Open the search field without logging into the application
2. Enter a valid product name in the search field
3. Click the search button
4. Observe the bottom section of the search results page

---

## ✅ Expected Result

The system must display the "Items per page" filter with available options such as 12, 24, 36, and 48 for unauthenticated users
