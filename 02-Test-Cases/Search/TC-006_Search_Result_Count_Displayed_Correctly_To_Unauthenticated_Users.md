# TC-0096 - Search Result Count Displayed Correctly to Unauthenticated Users

## 🧪 Test Steps
1. Perform a valid product search without logging into the application
2. Observe the result count displayed at the bottom of the page (example: "1–12 of 48")

---

## ✅ Expected Result

The system must correctly display the current result range and total number of search results for unauthenticated users
