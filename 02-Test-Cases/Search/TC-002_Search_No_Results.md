# TC-002 – Verify search with no matching results

## 🧪 Test Steps
1. Open the application  
2. Log in with valid credentials  
3. Locate the search bar  
4. Enter a non-existing product name (e.g., "xyz123invalid")  
5. Press "Enter" or click on the search icon  

---

## ✅ Expected Result
 
A message indicating no results is shown (e.g., "No results found")
